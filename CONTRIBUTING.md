# Contributing to Copilot Agents Dojo 🥋

Welcome to the dojo. Contributions that sharpen the disciplines are always welcome.

## How to Contribute

1. **Fork the repo** and create a feature branch.
2. **Follow the dojo rules**: use `tasks/todo.md` to plan your change, run `scripts/verify.sh` before submitting.
3. **Open a PR** with a clear description of what you changed and why.

## What We're Looking For

- **New disciplines or refinements** to existing ones in `skills.md`
- **Stack-specific examples** for `copilot-instructions.md` (e.g., Rust, Ruby, Kotlin)
- **Helper scripts** that reduce token burn and automate repetitive agent tasks
- **Automation improvements** — GitHub Actions, linters, or tooling that enforces the dojo rules
- **Real-world lessons** — if you've trained agents with the dojo and learned something, share it

## Guidelines

- Keep changes focused. One concern per PR.
- Test scripts locally before submitting.
- Don't over-engineer — the dojo values simplicity.
- Be respectful. We're all here to learn.

## Reporting Issues

Open an issue with:
- What you expected
- What happened
- Steps to reproduce (if applicable)
